• Static charts

    <nvd3 options="options" data="data"></nvd3>

    Any data changing doesn't affect the chart. It's default directive behaviour [v1.0.2+].

  • Dynamic charts (only data update)

    <nvd3 options="options" data="data" config="{deepWatchData: true}"></nvd3>

    Directive watches for data changing and automatically updates the chart. Note that in this case the angular $watch procedure is used for tracking data, that may lead to memory leak for large datasets or multiple charts on the page. It's a convenient way for lightweight dynamic charts.

    An alternative and more reliable solution is using api attribute: 

    <nvd3 options="options" data="data" api="api"></nvd3>
    and then update the chart in javascript when needed 
    $scope.data = ... // set new data
$scope.api.update();

  • Dynamic charts (complete refresh)

    <nvd3 options="options" data="data" config="{deepWatchData: true, refreshDataOnly: false"></nvd3>

    Directive watches for data changing and automatically refresh the chart. Note that in this case the angular $watch procedure is used for tracking data, that may lead to memory leak for large datasets or multiple charts on the page. It's a convenient way for lightweight dynamic charts.

    An alternative and more reliable solution is using api attribute: 

    <nvd3 options="options" data="data" api="api"></nvd3>
    and then refresh the chart in javascript when needed 
    $scope.data = ... // set new data
$scope.api.refresh();
